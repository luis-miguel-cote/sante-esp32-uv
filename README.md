# 🌞 ESP32 UV Monitoring Device (Secure IoT Firmware)

## 📌 Overview

This repository contains the firmware for an ESP32-based IoT device designed to send UV radiation data securely to a cloud backend.

The device communicates over HTTPS using TLS (mbedTLS with Root Certificate Bundle) and sends periodic sensor readings to a Node.js backend connected to a MySQL database.

> ⚠️ The physical UV sensor integration is currently pending.  
> The communication layer and cloud integration are fully functional.

---

## 🏗 System Role

This repository covers only the embedded firmware layer:

ESP32 (ESP-IDF + FreeRTOS)  
↓ HTTPS (TLS secure connection)  
Node.js Backend (Cloud)  
↓  
MySQL Database (UTC timestamps)

---

## 🔐 Security Features

- HTTPS secure communication
- TLS certificate validation using ESP-IDF certificate bundle
- No insecure certificate bypass
- API Key authentication for device identification
- Server-side timestamp control (UTC-based architecture)

---

## ⚙️ Technical Details

- Framework: ESP-IDF
- Environment: PlatformIO
- RTOS: FreeRTOS
- TLS: mbedTLS with `crt_bundle_attach`
- Communication: HTTPS POST requests
- Transmission interval: Every 3 minutes
- Data format: JSON

Example payload:

```json
{
  "api_key": "DEVICE_API_KEY",
  "valor_uv": 7.5
}
```

---

## 🧠 Engineering Design Decisions

- The device does not generate official timestamps.
- Time synchronization is handled via SNTP.
- All official timestamps are generated by the backend (UTC).
- Secure production-ready TLS implementation (no insecure mode).
